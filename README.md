# commentsAuth
## comments API 인증 서버
- 소셜 로그인으로 사용자를 로그인시킵니다.
- 닉네임, 프로필사진, 고유 ID와 [CommentsAPI](https://github.com/ynifamily3/CommentsApi)에서의 검증을 위한 `jwt` 키를 생성해서 [commentsFront](https://github.com/ynifamily3/CommentsFront)에게 전달해 줍니다.
